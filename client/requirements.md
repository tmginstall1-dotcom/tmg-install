## Packages
framer-motion | Beautiful page transitions, spring animations, and micro-interactions
date-fns | Formatting dates for bookings and timelines
react-day-picker | Beautiful calendar component for selecting booking slots
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes without style conflicts

## Notes
The application uses cookie-based auth (credentials: "include") for admin/staff.
The API endpoints are defined in @shared/routes.
Images are assumed to be dynamic (e.g., photoUrls from staff), so they use standard img src tags.
Stock images are from Unsplash for the landing page hero.
